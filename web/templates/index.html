CBZify Help & Documentation

Uploading Files

CBZify supports PDF and EPUB comic files up to 2GB each.

  • Drag & Drop: Drop files directly onto the upload area
  • Browse: Click "Choose Files" to select from your device
  • Multiple Files: Upload several files at once for batch processing
Conversion Settings
  • DPI: Image resolution (50-600, default: 300)
  • Format: PNG (lossless), JPEG (small), or WebP (modern)
  • Quality: JPEG/WebP compression (1-100, default: 95)
  • Workers: Parallel threads (1-6, default: 4)
  • Skip Analysis: Skip content analysis for consistent speed
Smart Processing

PDF Analysis: CBZify automatically detects the best conversion method:

  • Text Overlays Detected: Renders complete pages to preserve speech bubbles and captions
  • Image-Only PDFs: Extracts images directly for lossless quality
  • Skip Analysis: Always renders pages for consistent performance
Downloads

Converted CBZ files are ready for download immediately after conversion completes.

Docker Usage

For self-hosting, use our secure distroless container:

docker run -p 8080:8080 ghcr.io/skgchp/cbzify:latest
About CBZ Format

CBZ (Comic Book Zip) is a widely-supported format for digital comics. It's simply a ZIP archive containing sequential image files that comic readers can display page by page.
